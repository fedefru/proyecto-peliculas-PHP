# Proyecto Películas 

Proyecto para el instituto en el que cargo y visualizo peliculas. Incluye paginado, exportación del listado a PDF, trigger, procedimientos almacenados y JQuery.

## Instalación

Una vez que haya instalado el XAMPP puede insertar el proyecto allí y arrancar APACHE y MYSQL. Dentro del archivo va a encontrar la base de datos que debera importar a Phpmyadmin.
